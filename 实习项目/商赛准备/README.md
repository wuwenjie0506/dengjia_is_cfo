┌──────────┐

- 提前组队比临时凑人的团队强 10 倍

## 备赛工具箱

### 案例分析框架

```
拿到一个案例后的分析步骤：

1. 公司/行业背景（1h）
   └── AI 辅助：让 Claude 帮你搜集行业概况、竞品情况

2. 财务分析（3-4h）
   ├── 三张表关键指标提取
   ├── 盈利能力、偿债能力、营运能力
   └── 现金流健康度判断

3. 问题诊断（2-3h）
   ├── 用 SWOT / 波特五力 做定性
   └── 用数据支撑每一个判断（关键！）

4. 方案设计（4-5h）
   ├── 每个方案必须有：what + why + how + 财务测算
   └── 方案之间的权衡比较

5. PPT 制作（4-5h）
   ├── 结论先行，每页一个核心观点
   └── 数据可视化 > 文字堆砌

6. 演讲排练（3-4h）
   └── 控制时间，准备 Q&A 可能被挑战的点
```

### AI 辅助流程

```
阶段              AI 做什么                    你做什么
───────────────────────────────────────────────────
行业调研    →   搜集行业规模、增速、政策     →   判断哪些信息有用
财务分析    →   快速拉指标、发觉异常数据     →   解释异常的原因
方案生成    →   给出 3-5 个方案方向          →   筛选、深化、做财务测算
PPT        →   给每页的文案框架             →   调整逻辑、设计呈现
Q&A 模拟   →   扮演评委提问                 →   打磨回答
```

## 日常训练（每周）

| 训练项 | 频率 | 方法 |
|--------|------|------|
| **案例速读** | 每周 1 个 | CIMA 官网有历年真题，下载来快速过 |
| **财务快析** | 每周 1 家 | 拿到一家陌生公司的报表，30min 内找出 3 个核心问题 |
| **演讲训练** | 每周 1 次 | 对着手机录 5 分钟分析陈述，自己回看挑毛病 |
| **行业积累** | 日常 | 关注新能源、消费、科技三个赛道的基础知识 |
