# 项目三：自媒体——垂直的财会成长 Vlog

## 定位

```
定位公式：
大二会计生 + 正在冲向四大/CFO + 真实的成长记录 + 专业财报研读

人设：不是"教你的老师"，而是"和你一起成长的人"
```

## 平台选择

| 平台 | 内容形式 | 优势 | 投入 |
|------|---------|------|:--:|
| **小红书** | 图文 + 短视频（<3min） | 财会/大学生人群密，适合起步 | ⭐⭐ |
| **B站** | 中长视频（5-10min） | 内容深度高，长尾流量好 | ⭐⭐⭐ |
| **公众号** | 深度图文 | 适合财报研读长文 | ⭐⭐⭐ |
| **抖音** | 短视频（<1min） | 流量大但泛人群，变现难 | ⭐⭐⭐ |

> **起步建议**：小红书为主阵地（图文门槛低）+ B站同步发（财报研读视频版）

## 执行节奏

```
Month 1：试水期
├── 每周 2 条 Vlog + 1 篇财报研读图文
├── 目标：找到自己的表达风格
└── 指标：粉丝 0→200

Month 2：迭代期
├── 看数据（赞藏评），砍掉没人看的内容类型
├── 加大反馈好的内容
└── 指标：粉丝 200→800

Month 3：稳定期
├── 固定栏目节奏
├── 开始做第 1 条爆款选题（蹭热点+专业解读）
└── 指标：粉丝 800→2000，至少 1 条笔记赞藏过 1000
```

## AI 辅助内容生产

```
选题策划   → Claude：每周抛 10 个选题，你挑 3 个
文案框架   → Claude：给初稿框架，你改写加个人观点
标题优化   → Claude：给 10 个标题备选，你选最优
封面制作   → Canva AI / 醒图 AI 抠图
视频剪辑   → 剪映 AI（自动字幕、AI 配音占位）
数据复盘   → 人工看后台，AI 帮你总结规律
```

> ⚠️ 红线：AI 可以帮你写框架，但**文字必须经过你改写**。真实感和个人声音是 Vlog 的核心竞争力。
