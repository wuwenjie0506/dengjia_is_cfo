# 第一份实习：准备、难点与突破路线

> 写在前面：第一份实习是整条路最难的——因为你什么都没有，而招聘方想要什么都有的人。但突破第一份之后，后面就是正循环。

---

## 一、第一个残酷现实：你不是在和同学竞争

```
你的真实竞争对手：
┌────────────────────────────────────────────┐
│ 大二/大三已经有一段实习的人              │  ← 滚雪球效应，有实习的人更容易拿到下一份
│ 家里安排好了的同学                        │  ← 不走公开招聘通道
│ 名校 target school 自带光环               │
│ 大四/研一来降维打击的低年级岗              │
└────────────────────────────────────────────┘
```

**核心难点**：简历上什么都没有，HR 凭什么筛出你？

**突破逻辑**：用**"可展示的成果"**替代**"工作经历"**。你没有实习，但你可以有作品。

## 三、没有实习经历，就拿"作品"替代

### 方案 A：上市公司深度财务分析报告（最推荐）

拿一家你感兴趣的真实上市公司（茅台、比亚迪、泡泡玛特、瑞幸……），做一份 10-15 页的财务分析报告：

```
报告结构（可以直接当作品集附件投递）：
1. 公司业务概览（1-2 页，行业地位、收入构成）
2. 三张表核心指标拆解（3-4 页，收入质量、利润含金量、现金流健康度）
3. 杜邦分析 + ROE 拆解（2 页）
4. 同行对比（2-3 页，选 2-3 家可比公司横向对比）
5. 估值（1-2 页，PE/PB 历史的纵向 + 同行横向）
6. 风险点（1 页，应收账款、商誉、关联交易……）
```

**做这个报告，你在面试时能说：**

> "我虽然没有实习经历，但我独立完成了茅台 2023-2025 年三年的财务分析报告，做了杜邦拆解和可比公司对比分析，这是我做的报告。"

这句话比"我学习能力强，希望能得到锻炼"强 100 倍。

### 方案 B：商业竞赛

首选：**CIMA 商业精英挑战赛** / **华为财务精英挑战赛** / **QP 个案分析比赛**

- 组团参加，不需要单人搞定
- 拿了奖直接写进简历
- 没拿奖，过程也可以写成项目经历
- 2026 年开始用 AI 辅助做行业研究和数据可视化，效果会远超纯手工组

### 方案 C：自媒体 / 内容输出

在小红书 / 公众号 / B站上持续输出财务/经济类内容：

- "大二会计生拆解蜜雪冰城财报"系列
- 持续更新 30 天，就是一份动态作品集
- 面试官会觉得你有表达能力和主动性

## 四、第一份实习投哪？（按难度和 ROI 排序）

| 梯队 | 目标 | 难度 | 策略 |
|:--:|------|:--:|------|
| **T3** | 本地中小会计师事务所 / 代账公司 | ⭐ | 保底，寒假年审期间大量缺人 |
| **T2** | 国内所：立信、天健、信永中和等 | ⭐⭐ | 门槛低于四大，业务量巨大，年审疯狂招实习生 |
| **T1** | 企业财务部：大厂/中型企业财务实习生 | ⭐⭐⭐ | 岗位少、竞争激烈，但成长快 |
| **T0** | 四大：PwC/德勤/安永/毕马威 | ⭐⭐⭐⭐ | 大三寒假的常规路径 |

### 第一份实习的最优策略

```
不要死磕四大。第一份先拿 T2/T3 入场券。
路径：T3（大二暑假）→ T2（大三寒假）→ T1/T0（大三暑假）
经历过第一份，简历就有人看了。
```

## 五、每个环节的难点和突破方法

### 难点 1：简历石沉大海

**根因**：你的简历关键词没命中筛选条件。

**突破**：

- 简历上必须出现这些词：**Excel 高级函数、Power Query、财务报表分析、杜邦分析、行业研究**
- 不要写"熟练掌握 Office"，改成"用 Power Query 处理过 50 万行财务数据并输出分析报告"
- 不要写"有团队合作精神"，改成"带队参加 CIMA 商赛，负责财务建模模块"
- 用 AI（ChatGPT/Claude）帮你改一版简历，把每句话从"描述职责"变成"展示成果"

### 难点 2：没有面试机会

**根因**：你只在招聘网站上等。

**突破——三条非主流渠道**：

| 渠道 | 怎么做 | 效果 |
|------|--------|------|
| **学长学姐内推** | 在领英/微信上找本专业去四大/企业工作的学长学姐，约 15 分钟线上聊聊，别一上来就要内推 | 成功率最高 |
| **宣讲会/校园大使** | 各大所校宣讲会必去，现场加 HR 微信，申请"校园大使" | 有面熟加成 |
| **直接投带教老师** | 在脉脉/小红书上找到四大/国内所的 senior associate，礼貌私信问组里招不招实习生 | 绕过 HR 筛选 |

### 难点 3：面试被问"你没有经验，我们要你有什么用"

**面试官真实意思**：我需要一个能干活、不出乱子、不给我添麻烦的人。

**回答公式**：承认没有经验 → 展示你已经做了什么准备 → 证明你能快速上手

**示例**：

> "确实我目前没有正式的实习经历。但为了弥补这个差距，我过去两个月做了三件事：第一，我完整分析了比亚迪近三年的财务报表，做了一份杜邦分析和同行业对比报告，熟悉了财务分析的逻辑；第二，我自学了 Excel Power Query 和基础 SQL，可以用数据透视表快速处理大量数据；第三，我在学校的会计专业课排名前 20%，底稿的借贷逻辑我不会有问题。所以虽然是第一次实习，但我能比别人更快上手具体工作。"

### 难点 4：拿到了实习但完全不会做事

**真实情况**：第一周就是蒙的，所有人都这样。

**快速上手指南**：

- 带个本子，带教说的每句话都记下来。说一遍记不住很正常，但问第二遍就不好了
- 领到任务时确认三件事：**什么时候要 + 格式要求 + 有没有去年的模板可以参考**
- 底稿做完后自己检查一遍，钩稽关系对得上再交
- 主动问有没有其他可以帮忙的，但不要在不忙的时候玩手机
- 把 AI 用起来：不会写的分析段落先让 AI 给个框架，底稿公式写不出让 AI 帮你 debug

## 七、心态底线

| 真相 | 应对 |
|------|------|
| 投 20 份，15 份没回音 | 正常，不是你的问题 |
| 面试被拒 | 每次面试都是免费模拟考，回来复盘 |
| 同学靠关系去了好地方 | 跟你没关系，你的牌要自己打 |
| 第一份实习只能去小所 | 小所年审两个月比上课两年学得多 |
| 大二找不到实习 | 正常。大二暑假找不到就做项目 + 学技能，大三寒假一定有机会 |

**核心公式**：

```
第一份实习 = 70% 准备（作品 + 技能）+ 20% 渠道（内推 + 非主流路径）+ 10% 运气
```

运气不可控，但 90% 的事你可以现在就动手。
