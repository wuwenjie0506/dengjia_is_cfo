# 财务会计 · 经济学 · 金融学：关系、视角与课程书单

> 写给自己：你已是会计学专业，这篇重点补**金融学**和**经济学**的课程全景与学习路径。

---

## 一、一句话区分

| 学科 | 核心问题 | 一句话 |
|------|---------|--------|
| **经济学** | 资源怎么分配最有效？ | 研究"**为什么**"——为什么涨价，为什么失业 |
| **财务会计** | 钱花哪了、赚了多少？ | 记录"**是什么**"——已经发生了什么 |
| **金融学** | 钱放在哪最值钱？ | 决策"**怎么办**"——怎么融资、怎么投资 |

---

## 二、三者关系图

```
                    ┌───────────────┐
                    │    经济学      │  ← 顶层：解释规律，提供分析框架
                    │  (Why 为什么)  │
                    └───────┬───────┘
                            │ 供给需求、市场结构、货币政策
                            ▼
            ┌───────────────────────────────┐
            │          金融学                │  ← 中层：资源配置，资本决策
            │      (How 怎么办)              │
            │  投资、融资、定价、风险管理      │
            └───────────────┬───────────────┘
                            │ 估值模型、资本成本、财务报表分析
                            ▼
                    ┌───────────────┐
                    │   财务会计     │  ← 底层：数据基础，记录事实
                    │  (What 是什么) │
                    └───────────────┘
```

- **经济学给金融学提供理论底座**（利率由什么决定、市场为什么会波动）
- **财务会计给金融学提供数据燃料**（没有报表，估值和投资决策无从谈起）
- **金融学是经济学理论在企业资本决策中的落地**

> 类比：经济学是物理学（研究规律），财务会计是测量工具（记录数据），金融学是工程学（用规律和数据造东西）。

## 三、企业和政府的不同视角

### 企业视角

| | 经济学 | 财务会计 | 金融学 |
|------|--------|---------|--------|
| **关注点** | 行业景气度、供需格局、价格弹性 | 收入、成本、利润、现金流、资产负债 | 融资成本、投资回报、并购估值、汇率风险 |
| **典型问题** | 涨价客户会流失多少？新市场的规模多大？ | 这个季度收入确认对了吗？费用有没有少记？ | 该发债还是增发？这个项目 IRR 够不够门槛？ |
| **使用场景** | 战略规划、市场进入决策 | 日常核算、纳税申报、对外披露 | 预算分配、融资方案、投资者沟通 |
| **角色** | 参谋长（看大势） | 账房先生（记好账） | 金主/管家（管好钱） |

**企业侧重点**：会计是基本功，金融是进阶武器，经济学提供宏观视野。

---

### 政府视角

| | 经济学 | 财务会计 | 金融学 |
|------|--------|---------|--------|
| **关注点** | GDP 增长、就业率、通胀、产业结构 | 财政收入、财政支出、政府债务、社保基金 | 货币发行、利率调控、汇率管理、金融监管 |
| **典型问题** | 降息能刺激多少经济？财政赤字会不会推高通胀？ | 今年税收完成率多少？地方债还还得起吗？ | 要不要干预汇市？银行资本充足率够不够？系统性风险多大？ |
| **使用场景** | 制定货币/财政政策、五年规划 | 编制财政预算、审计政府账目 | 央行货币政策、金融稳定监管 |
| **核心部门** | 发改委、统计局、智库 | 财政部、审计署 | 央行、银保监会、证监会 |

**政府侧重点**：经济学是核心决策语言，财务会计管"国家账簿"，金融学管"国家钱袋子"。

## 四、金融学：完整课程体系

> 以国内金融学本科 + 国际对标为标准，标注推荐优先级。

### 4.1 核心必修（金融学专业的主干课）

| 课程 | 核心内容 | 优先级 | 对标英文课 |
|------|---------|:----:|-----------|
| **公司金融 / 公司理财** | 资本预算、资本结构、股利政策、WACC | ⭐⭐⭐ | *Corporate Finance* (Berk & DeMarzo) |
| **投资学** | 资产配置、CAPM、有效市场、组合理论 | ⭐⭐⭐ | *Investments* (Bodie, Kane, Marcus) |
| **金融市场学** | 一级/二级市场、金融中介、市场微观结构 | ⭐⭐⭐ | *Financial Markets and Institutions* (Mishkin) |
| **国际金融** | 汇率决定、国际收支、蒙代尔-弗莱明模型 | ⭐⭐⭐ | *International Finance* (Krugman) |
| **货币银行学** | 货币创造、央行调控、利率传导 | ⭐⭐⭐ | *The Economics of Money, Banking & Financial Markets* (Mishkin) |
| **金融工程 / 衍生品** | 期权期货定价、BS 模型、对冲策略 | ⭐⭐ | *Options, Futures, and Other Derivatives* (Hull) |
| **商业银行经营管理** | 资产负债管理、信贷风险管理、巴塞尔协议 | ⭐⭐ | *Bank Management & Financial Services* (Rose) |
| **风险管理** | VaR、信用风险、操作风险 | ⭐⭐ | *Risk Management and Financial Institutions* (Hull) |
| **固定收益证券** | 债券定价、久期凸性、利率期限结构 | ⭐⭐ | *Fixed Income Securities* (Tuckman) |
| **行为金融学** | 认知偏差、非理性市场、前景理论 | ⭐ | *A Behavioral Approach to Asset Pricing* (Shefrin) |

### 4.2 金融计量/数据分析（AI 时代极其重要）

| 课程 | 核心内容 | 优先级 |
|------|---------|:----:|
| **金融计量经济学** | 时间序列（ARIMA、GARCH）、面板数据、协整检验 | ⭐⭐⭐ |
| **金融数据处理 (Python)** | pandas/numpy 处理行情数据、财务数据、回测框架 | ⭐⭐⭐ |
| **金融机器学习** | 用 ML 做信用评分、欺诈检测、因子挖掘 | ⭐⭐ |
| **量化投资** | 多因子模型、Alpha 策略、回测 | ⭐⭐ |

## 五、经济学：完整课程体系

> 按"微观经济学 → 宏观经济学 → 计量经济学 → 应用领域"四层展开。

### 5.1 核心必修

| 课程 | 核心内容 | 优先级 |
|------|---------|:----:|
| **微观经济学（初/中级）** | 消费者厂商理论、市场结构（完全竞争/垄断/寡头）、一般均衡 | ⭐⭐⭐ |
| **宏观经济学（初/中级）** | GDP 核算、IS-LM、AD-AS、增长模型（索洛）、经济周期 | ⭐⭐⭐ |
| **计量经济学** | 回归分析（OLS）、假设检验、内生性处理（IV、DID、RDD） | ⭐⭐⭐ |
| **货币银行学** | 货币供需、通胀理论、货币政策传导机制 | ⭐⭐ |
| **国际经济学** | 比较优势、H-O 模型、关税与非关税壁垒、汇率 | ⭐⭐ |
| **财政学 / 公共经济学** | 税收理论、公共品、财政分权、社会保障 | ⭐⭐ |
| **政治经济学** | 剩余价值、资本积累、利润率下降 | ⭐（国内必修） |

### 5.2 应用/中级课程（按兴趣选修）

| 课程 | 核心内容 | 适合谁 |
|------|---------|--------|
| **产业经济学** | SCP 范式、定价策略、反垄断 | 想做行业分析、战略咨询 |
| **发展经济学** | 贫困陷阱、制度与增长、中等收入陷阱 | 想去世行/政府/智库 |
| **制度经济学** | 产权、交易成本、合约理论（诺斯+科斯+威廉姆森） | 想理解中国经济的制度逻辑 |
| **博弈论与信息经济学** | 纳什均衡、信号传递、委托代理、逆向选择 | 想走学术或做机制设计 |
| **劳动经济学** | 人力资本、工资决定、歧视 | 想做人力咨询、政策研究 |
| **区域/城市经济学** | 集聚、地租、Tiebout 模型 | 想做房地产/城市规划 |
| **环境经济学** | 外部性、碳定价、排污权交易 | 想做 ESG/绿色金融 |
| **中国经济专题** | 改革开放、分税制、土地财政、双轨制 | 每个中国商科学生都应该学 |

## 六、数据与分析工具（AI 时代，与课程同等重要）

> 你不是计算机专业，不需要当程序员。但以下工具是未来财务人的基本配置。

### 6.1 三层能力金字塔

```
         ┌─────────────┐
         │ AI 辅助分析  │  ← ChatGPT/Claude + Cursor 做深度分析
         │  & 决策支持  │
         ├─────────────┤
         │   BI 可视化   │  ← Power BI / Tableau 做 Dashboard
         ├─────────────┤
         │  数据处理    │  ← Excel(PQ) + SQL + Python(pandas)
         └─────────────┘
```

### 6.2 具体工具与何时学习

| 工具 | 用途 | 何时学 | 怎么学 |
|------|------|--------|--------|
| **Excel Power Query + Power Pivot** | 自动化数据处理，百万行级别 | 现在 | B站搜"Power Query 财务"，两周上手 |
| **SQL** | 从数据库取数，财务系统都在数据库上 | 大二暑假 | *SQL for Data Analysis* (Tanimura)，配合 LeetCode SQL 刷题 |
| **Python (pandas + matplotlib)** | 批量处理财务数据、自动化分析 | 大二暑假-大三 | 《利用 Python 进行数据分析》(McKinney) |
| **Power BI / Tableau** | 财务 Dashboard、经营分析看板 | 大三 | 拿真实财报数据做一个公司 Dashboard |
| **AI 辅助工具链** | Cursor 写代码、Claude 做行业分析、AI 审合同 | 现在 | 日常使用，当成生产力标配 |

### 6.3 数据源清单（培养"看数据"的习惯）

| 数据源 | 内容 | 用法 |
|------|------|------|
| **国家统计局** | GDP、CPI、PPI、PMI | 每月关注核心宏观指标 |
| **央行官网** | M2、LPR、社融、外汇储备 | 理解货币政策走向 |
| **Wind / Choice 终端** | 上市公司财务数据、行业数据库 | 学校图书馆一般有，学会用 |
| **东方财富 Choice** | A 股财报、估值数据 | 免费版够本科用 |
| **FRED (美联储数据库)** | 美国利率、就业、GDP | 全球最重要的免费宏观数据库 |
| **CSMAR / CNRDS** | 学术级中国上市公司数据 | 做研究/写论文用 |
| **巨潮资讯网** | A 股年报/公告原文 | 读一手年报，不要只看别人分析 |
| **SEC EDGAR** | 美股年报 10-K/10-Q | 英文财报实战 |

## 八、大二开始的学习路线

```
本学期（大二下）
├── 曼昆《经济学原理》→ 经济学的整体框架
├── 米什金《货币金融学》→ 金融世界怎么运转
└── Excel Power Query → 数据处理效率翻倍
└── 开始用 ChatGPT/Claude 辅助学习和写报告

大三上
├── 罗斯《公司理财》前 15 章 → 企业金融核心
├── 兰小欢《置身事内》→ 理解中国政府和经济
├── SQL 入门 → 学会从数据库取数
└── 读 3 份真实年报（茅台 / 比亚迪 / 一家你感兴趣的公司）

大三下
├── Bodie《投资学》→ 资产定价和组合管理
├── 徐高《宏观经济学二十五讲》→ 把理论和中国实践连接
├── Python (pandas) → 财务数据处理入门
└── 订阅 Damodaran 博客 + 做一个公司的完整估值模型

大四
├── Hull《衍生品》选读 → 对金融工具有认知
├── 佩因曼《财务报表分析与证券估值》→ 打通会计与金融
├── Power BI / Tableau → 做一个财务分析 Dashboard
└── 持续读年报 + 跟踪宏观数据
```

---

> **三句话总结**：  
> 经济学让你的大脑学会看"局"——看大势。  
> 财务会计让你的眼睛学会看"账"——看事实。  
> 金融学让你的手学会管"钱"——做决策。  
> **CFO 的日常 = 用会计的数据 + 经济学的框架 + 金融学的工具 + AI/数据的效率，帮 CEO 做判断。**
